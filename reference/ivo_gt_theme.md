# A nice GT theme

Applies a predefined theme to a GT table, including font, colors, and
alignment.

## Usage

``` r
ivo_gt_theme(table, color = "darkgreen", font_name = "Arial")
```

## Arguments

- table:

  A `gt` object.

- color:

  A named color or a color HEX code, used for the lines in the table.
  Defaults to "darkgreen".

- font_name:

  The name of the font to be used in the table. Defaults to "Arial".

## Value

A styled `gt` table.
